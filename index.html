Making songs available via Coria

This is a minimal demo of two ways to make practice material available for your choir via Coria.

For both of these two ways you need to have a website of some kind and of course you need to have the song or songs that you want to share in format that is supported by coria. The website can be very simple. This example uses github pages, which is free. But if your choir has its own website you should be able to use that. The formats that are supported by coria are musicXML and Noteworthy Composer format. A popular free option to create musicXML is Musescore. Many musicXML files for choral music can be downloaded from the site cpdl.org. Noteworthy composer files can be created using the tool Noteworthy Composer.
